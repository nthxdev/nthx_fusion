COMPONENT PATTERNS (see spec for exact styling):

- Dropdowns: Arrow rotates, list scrolls if >5 items, click-outside closes
- Sliders: Red track for active, white thumb, real-time input field
- Checkboxes: ☐ unchecked, ☑ checked in red
- Buttons: Primary #ff4444, Secondary #2a2a2a, hover #3a3a3a
- Drag-drop: Hover border #ff4444, accepts media types

BEHAVIOR SPECIFICS:
- Processor selection (radio click) → fade out old panel (150ms), fade in new panel (150ms)
- Dropdown open → arrow rotates (arrow-down ▼ to arrow-up ▲), list slides down 200ms
- Slider drag → show value tooltip, real-time preview updates (debounce 300ms)
- Media drop → show filename + size, enable Preview button
- Face selector → grid of thumbnails, click to select (white border + highlight)
- Progress during processing → show step (e.g., "3/12"), percentage bar, ETA
- Logs → new messages fade in, auto-scroll if at bottom, color-code by severity

ERROR HANDLING:
- No source media → START button disabled + message
- Model not downloaded → Warning badge + "Download?" option
- GPU unavailable → Suggest CPU mode
- Invalid output path → Error banner + directory creation prompt
